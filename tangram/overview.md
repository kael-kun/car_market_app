# Used Car Marketplace

## Core Concept
A web platform connecting used car sellers and buyers to list, browse, compare, and purchase second-hand vehicles efficiently and safely. Built with AstroJS, TypeScript, and Astro Actions for a performant, island-based architecture, the app uses Supabase Client JS exclusively for authentication, Prisma for database migrations and querying, Supabase for backend services, and Cloudflare R2 for object storage. Deployed on Cloudflare Workers for global low-latency access, the platform prioritizes transparent vehicle information, trust-building via verification and ratings, and streamlined buyer-seller communication.

## Target Audience
Primary users include car buyers seeking affordable, well-documented second-hand vehicles, and car sellers (individuals or dealerships) listing vehicles for sale. Secondary users are platform administrators responsible for listing moderation, user management, and maintaining platform trust and safety.

## High-Level Vision
From a user perspective, the app feels fast, intuitive, and trustworthy: sellers can quickly create rich, image-backed listings with detailed vehicle specs, while buyers can filter and search for cars by price, location, brand, and more, save favorites, compare options, and directly contact sellers. The MVP focuses on core workflows (authentication, listing management, search/filtering, basic messaging), with future expansions for ratings, analytics, and advanced features.
